Overview

About

Text code is a simple project that i created to make writing basic code as easy/readable as possible.

How it works

Text Code uses keywords to easily generate code at runtime, every keyword is case-insensitive, and every word seperated by spaces that isnt a valid keyword will be treated as variables.

Keywords

Keywords are very important for Text Code, they tell the compiler what to do with the input text.

See a list of all keywords below.

Add, Subtract, Divide, Multiply, From, To, Is, Are

Variables

In Text Code every things that isnt a reserved keyword is automatically treated as a variable.

Example:

See defining variables for info on how to do so

x is 3; add
hi 4

Defining Variables

Defining variables in Text Code is very easy, simply write the name you want your variable to be called and use the is keyword to assign a value to it.

Examples:

Assign a number to a variable

x is 10;
y is 5;
x subtract y 5

Assign text to a variable

hi is "Hello";
name is " User";
hi name Hello User

Assign a list to a variable

You can use is and are interchangeably its only purpose is for readability, they both do the same thing.

also note that the spaces in between list value assignment do not matter.

numbers are 3, 5, 6
letters are "a","b","c"
numbers
letters 356
abc

Math

Text code makes it very easy to do complex math simply by telling it what to do in english.

Add

The add keyword simply works how anyone would expect, you can put it in between to numbers, or in front of the sequence of numbers you want to add.

Examples:

add 1, 3 4 1 add 3 4

the add keyword can also be used by the + symbol in the same places.

Subtract

The subtract keyword simply works how anyone would expect, you can put it in between to numbers, or in front of the sequence of numbers you want to subtract.

Examples:

subtract 10, 8 2 22 subtract 3 19

the subtract keyword can also be used by the - symbol in the same places

Divide

The divide keyword simply works how anyone would expect, you can put it in between to numbers, or in front of the sequence of numbers you want to divide.

Examples:

divide 20, 2 10 69 divide 3 23

the divide keyword can also be used by the / symbol in the same places

Multiply

The multiply keyword simply works how anyone would expect, you can put it in between to numbers, or in front of the sequence of numbers you want to multiply.

Examples:

multiply 10, 8 80 5 multiply 3 15

the multiply keyword can also be used by the * symbol in the same places
